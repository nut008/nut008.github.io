c常用函数

1.stdio.h函数库

1.1 字符输入输出函数

函数名 函数原型 函数功能 返回值
getchar int getchar() 从键盘缓冲区读取一个字符。 返回用户输入的字符。
fgetc和getc int fgetc(FILE *stream)
int getc(FILE *stream)		 这两个函数基本等价，从目标位置读取字符(参数为stdin时从终端读取)。 成功返回读取的字符，错误或文件尾返回EOF
gets char *gets(int *str) 将输入的字符串放到str数组里 操作成功返回数组指针，否在返回NULL
fgets char *fgets(char *str,int num,FILE *stream) 从目标位置读取num-1个字符或遇到回车，将读取的字符存到str中 操作成功返回str，遇到结束符返回空指针，若出错内容不确定。
putchar int putchar(int ch) 输出ch字符 成功时返回字符ch，否则返回EOF。
fputc和putc int fputc(int ch,FILE *stream)
int putc(int ch,FILE *stream)		 输出一个字符到目标位置 成功返回写入的字符，否则返回EOF（stdout输入到屏幕）。
puts int puts(const char *str) 在屏幕输出str指向的字符串 成功返回非负值，失败返回EOF
fputs int fputs(const char *str,FILE *stream) 将str输出到目标位置 成功返回一个非负数，失败返回EOF

标准输入文件（stdin），通常对应终端的键盘；标准输出文件（stdout）和标准错误输出文件（stderr），这两个文件都对应终端的屏幕。

1.2 数据块输入/输出函数

函数名 函数原型 函数功能 返回值
fread int fread(void *buffer,int size,int count,FILE *stream) 从目标位置读取count个字段，每个字段大小size个字节，存到buffer中 返回实际读取的字段个数
fwrite int fwrite(void *buffer,int size,int count,FILE *stream) 将buffer内容输出count个字段，每个字段大小size个字节到目标位置 返回实际写入的字段个数

1.3 格式化输入/输出函数

1.3.1 printf

格式控制符 说明
d或i 带符号的十进制整数
o 无符号八进制整数
x 无符号十六进制整数
u 无符号十进制整数
f 小数形式单、双精度浮点数
e或E 指数形式的单、双精度浮点数
g或G 以%f或%e形式输出浮点数
c 单个字符
s 字符串
p 输出一个指针
flags 说明 例子
- 在给定的域宽内靠左端输出 ‘%-10d’左对齐输出宽度为10（”%10d”右对齐）
+ 强制在正整数前输出+，负数前输出- “%+d”(与对齐一起使用”%+-10d”)
# 使用o,x,X时加前导0,0x,0X输出 “%#x”(与对齐一起使用”%#-10x”)
width 说明 例子
m 输出字段的宽度，如果数据的宽度小于m，则左端补上空格，否则按照实际位数输出 ‘%-10d’左对齐输出宽度为10（’%10d’右对齐）
precision 说明 例子
n 对于浮点数，表示输出n位小数；对于字符串，表示输出字符串的个数 “%.2s”输出字符串前两位，”%.2f”输出浮点数前两位

1.3.3 sprintf

函数原型 函数功能 返回值
int sprintf(char *buffer,const char * format[,argument]) 将format指向的格式化数据输出到buffer指向的数组中 返回输出的字符的个数
格式 功能 结果
sprintf(s, “%d”, 123); 把整数123 打印成一个字符串保存在s 中 产生：” 123”
sprintf(s, “%4d%4d”, 123, 4567); 指定宽度存储数据 产生：” 1234567”
sprintf(s, “%-4d%4d”, 123, 4567); 左对齐存储数据 产生：”123 4567”
sprintf(a,”%x”,13); 将13转换成16进制 产生：”d”

2.ctype.h函数库

函数名 函数原型 函数功能 返回值
isalnum int isalnum(int ch) 判断ch是否为十进制数或字母 若ch是英文字母或数字字符，返回非0值，否在返回0
isalpha int isalpha(int ch) 判断ch是否为英文字母 若ch为(‘A’~’Z’,’a’~’z’)返回非0，否则返回0
isdigit int isdigit(int ch) 判断ch是否为数字字符 若ch为(‘0’~’9’)返回非0值，否则返回0
iscntrl int iscntrl(int ch) 判断ch是否是控制字符(ASCII码0~31，或0x7f) 若ch是控制字符返回非0，否则返回0
isgraph int isgraph(int ch) 判断ch是否可打印字符(不包括空格) 若ch是可打印字符返回非0值，否则返回0
islower int islower(int ch) 判断ch是否是小写英文字母 若ch是小写英文字母返回非0，否则返回0
isupper int siupper(int ch) 判断ch是否是大写的英文字母 若ch是大写英文字母返回非0，否则返回0
isprint int isprint(int ch) 判断ch是否是可打印字符（包括空格） 若ch是可打印字符返回非0，否则返回0
ispunct int ispunct(int ch) 判断ch是否是标点符号 若ch是标点符号返回非0，否则返回0
isspace int isspace(int ch) 判断ch是否是空白符(\t,\n,\v,\f,\r,空格) 若ch是空白符返回非0，否则返回0
isxdigit int isxdigit(int ch) 判断ch是否是十六进制数字 若ch是十六进制数字对应字符返回非0，否则返回0
tolower int tolower(int ch) 若ch是’A’~’Z’的大写字母，此函数会返回ch小写形式 若ch是大写字母返回其小写字母，否则返回原字符
toupper int toupper(int ch) 若ch是’a’~’z’的小写字母，此函数会返回ch大写形式 若ch是小写字母返回其大写字母，否则返回原字符

3.1 字符串比较函数

函数名 函数原型 函数功能 返回值
memcmp int memcmp(const void *buf1,const void *buf2,unsigned count) 比较buf1和buf2的前count个字符的大小 buf1比buf2，小返回小于0，等返回0，大返回大于0
strcmp int strcmp(const char *str1,const char *str2) 比较字符串str1和字符串str2大小 str1=str2返回0，str1>str2返回一个大于0的值，str1<str2返回一个小于0的值
strncmp int strncmp(const char *str1,const char *str2) 与memcmp类似，二者区别：memcmp不检测’\0’，有可能越界 str1=str2返回0，’<’返回一个小于0的值，’>’返回一个大于0的值

8 string&replace(size_t pos1,size_t n1,size_t n2,char c) 用n2个字符c替换当前字符串对象中第post1个字符开始的n1个字符
9 string&replace(iterator i1,iterator i2,size_t n2,char c) 用n2个字符c替换当前字符串对象中迭代器i1和i2之间的字符
10 template<class InputIterator>string&replace(iterator i1,iterator i2,InputIterator j1,InputIterator j2) 用迭代器j1和j2之间的字符替换当前字符串对象中迭代器i1和i2之间的字符

返回值都为：字符串对象的引用
