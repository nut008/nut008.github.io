socket(familt,type[,protocol]) 构造socket对象。
s.bind(address) 将socket对象s绑定到指定地址，地址以元组(host,port)形式表示。

s.accept() 接受来自客户端的TCP连接请求，并返回元组(conn,address),conn是socket对象，address是客户端地址。
